# תיעוד פרויקט RepUp

## 1. סקירה כללית ומטרות (App Overview & Goals)
**RepUp** היא אפליקציית מעקב אימונים מתקדמת המיועדת לספק למתאמנים חווית משתמש פרימיום, נוחה ומעצימה.
המטרה העיקרית היא לאפשר למשתמשים לתכנן, לבצע ולעקוב אחר האימונים שלהם בצורה אינטואיטיבית, תוך שימוש בעיצוב "Neumorphic" מודרני ותמיכה מלאה בשפה העברית.

### מטרות מרכזיות:
- **פשטות:** ממשק משתמש נקי וברור המאפשר התמקדות באימון עצמו.
- **גמישות:** יכולת לבנות אימונים מותאמים אישית או להשתמש בתבניות.
- **מעקב:** שמירת היסטוריית אימונים מפורטת לצורך ניתוח התקדמות.
- **המשכיות:** יכולת להמשיך אימונים שהופסקו באמצע (Resume Workout).

---

## 2. פיצ'רים קיימים (Implemented Features)

### ניהול משתמשים
- **התחברות:** מערכת התחברות פשוטה מבוססת מספר טלפון.
- **פרופיל:** זיהוי משתמש והצגת שם בלוח הבקרה.
- **הרשאות אדמין:** גישה לממשק ניהול למשתמשים מוגדרים.

### לוח בקרה (Dashboard)
- **סטטיסטיקות:** הצגת מספר אימונים חודשי ושבועי.
- **אימון אחרון:** תצוגה מהירה של מועד וסוג האימון האחרון.
- **ניווט:** גישה מהירה לבניית אימון ולהיסטוריה.

### בונה האימונים (Workout Builder)
- **בחירת תרגילים:** סינון לפי קבוצות שרירים (חזה, גב, רגליים וכו').
- **חיפוש:** (מתוכנן/קיים חלקית) יכולת לאתר תרגילים ספציפיים.
- **הוספה לאימון:** יצירת רשימת תרגילים לאימון הנוכחי.

### אימון פעיל (Active Workout)
- **מעקב סטים:** הזנת משקל וחזרות לכל סט.
- **סימון ביצוע:** סימון תרגילים שהושלמו (V כחול).
- **היסטוריית תרגיל:** הצגת ביצועים מאימון קודם (משקל/חזרות) להשוואה מיידית.
- **מדיה:** צפייה בסרטוני הדגמה ותמונות של התרגיל.
- **שמירה אוטומטית:** מנגנון Auto-save למניעת אובדן נתונים.
- **ביטול אימון:** אפשרות לבטל אימון ולמחוק אותו מההיסטוריה (כולל מנגנון הגנה).
- **סיום וסיכום:** מסך סיכום חגיגי המציג כמות תרגילים, זמן וקלוריות.

### היסטוריית אימונים (Workout History)
- **רשימת אימונים:** צפייה בכל האימונים שבוצעו.
- **פירוט:** הרחבת אימון לצפייה בסטים ובמשקלים.
- **המשך אימון (Resume):** זיהוי אימונים חלקיים ואפשרות להמשיך אותם מאותה נקודה.
- **חזרה על אימון (Repeat):** שכפול אימון שהושלם לאימון חדש.
- **מחיקה:** הסרת אימונים מההיסטוריה.

### ממשק ניהול (Admin)
- **ניהול תרגילים:** הוספה, עריכה ומחיקה של תרגילים במערכת.
- **ייבוא/ייצוא:** תמיכה ב-CSV לניהול מאגר התרגילים.
- **תיקון נתונים:** כלים לסידור שמות שרירים ומיגרציית נתונים.

## 3. פיצ'רים מתוכננים - מצב מאמן (Coach Mode Architecture)
מערכת ה-Coach Mode נועדה לאפשר למאמנים לנהל מתאמנים מרחוק, או לשלב בינה מלאכותית (Apex AI) כמאמן וירטואלי.

### ארכיטקטורה מתוכננת:
1.  **ישות "מאמן" (Coach):**
    -   יכולת ליצור תוכניות אימון (Templates) ולשייך אותן למתאמנים.
    -   צפייה בלוח בקרה מרוכז של כל המתאמנים.
    -   קבלת התראות על סיום אימון של מתאמן.

2.  **ישות "מתאמן" (Trainee) - הרחבה:**
    -   שיוך למאמן ספציפי.
    -   קבלת "משימות" (אימונים לביצוע) ללוח הבקרה.
    -   דיווח מדדים (משקל גוף, היקפים) למאמן.

3.  **AI Coach (Apex AI):**
    -   ניתוח היסטוריית אימונים ליצירת המלצות להעלאת משקלים (Progressive Overload).
    -   בניית תוכנית אימונים אוטומטית על בסיס מטרות (חיטוב/מסה).
    -   משוב בזמן אמת במהלך האימון.

## 4. מבנה נתונים (Data Structure)

### User (משתמש)
```json
{
  "id": "string (UUID/Auto)",
  "firstName": "string",
  "lastName": "string",
  "phone": "string (Unique Identifier)",
  "isAdmin": "boolean",
  "createdAt": "timestamp"
}
```

### Exercise (תרגיל)
```json
{
  "id": "string",
  "name": "string",
  "mainMuscle": "string (e.g., 'Chest', 'Back')",
  "trackingType": "string ('weight', 'time', 'reps_only')",
  "video_url": "string (URL)",
  "imageUrls": ["string (URL)"]
}
```

### Workout Log (תיעוד אימון)
```json
{
  "id": "string",
  "userId": "string",
  "workoutName": "string",
  "timestamp": "timestamp",
  "status": "string ('completed', 'in_progress', 'partial')",
  "durationMinutes": "number",
  "calories": "number",
  "exercises": [
    {
      "exercise_id": "string",
      "name": "string",
      "mainMuscle": "string",
      "isCompleted": "boolean",
      "sets": [
        {
          "weight": "number/string",
          "reps": "number/string"
        }
      ]
    }
  ]
}
```

### Template (תבנית אימון)
```json
{
  "id": "string",
  "userId": "string (or null for global)",
  "name": "string",
  "exercises": [
    {
      "id": "string",
      "name": "string"
    }
  ]
}
```
